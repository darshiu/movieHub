## 🛠️ Installation

1️⃣ Clone the repository:

```bash
git clone https://github.com/darshiu/movieHub
```

2️⃣ Navigate to project directory:

```bash
cd movieHub
```

3️⃣ Install dependencies:

```bash
npm install
```

4️⃣ Run development server:

```bash
npm run dev
```

5️⃣ Open in browser:

- Visit [http://localhost:3000](http://localhost:3000)
